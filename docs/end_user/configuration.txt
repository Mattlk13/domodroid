=============
Configuration
=============

In the menu view you have choice of:

* Sync

Just run a mechanism to sync your domogik configuration with domodroid.

If it fail, start to check if rinor is listening on the good adress and port, not on localhost.

Be careful, it will destroy all your cutom_name and delete map widget link. It means you have to re-create you map design with your widget.
