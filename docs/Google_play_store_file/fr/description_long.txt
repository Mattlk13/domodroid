Domodroid est une application de contrôle à  distance qui vous permet de contrôler votre systà¨me domotique  via Domogki.

Les fonctions suivante sont déjà  disponibles dans cette version de Domodroid:

- Courbe avec un moteur de tracé interne (par défaut)
- Courbe avec le moteur Achartengine (en cours de développement)
- Visualisation des caméras
- Widgets On/Off ( deux versions disponibles), variateur, état et données, envoie de commande
- Mise à  jour automatique des widgets
- De nombreuses icônes
- Technologie supportée: Onewire, PLCBus, IPX800, Mirror, Teleinfo, X10, WakeOnLan, Ping, Zibase, RFXCom, YahooWather. (Toutes celles supportées par Domogik en théorie)
- Toutes tailles d'écrans téléphones et tablettes
- Contrôle de la mise en veille de l'appareil
- Démarrage automatique en mode map
